Обработка ошибок. Исключения.

Обработка исключений в Python осуществляется с помощью конструкции try-except, которая позволяет программе обрабатывать исключительные ситуации, возникающие во время выполнения кода. Кроме того, существует также конструкция finally, которая выполняется всегда, независимо от того, возникло исключение или нет.

Пример использования конструкции try-except:

try:
 result = 10 / 0 # Попытка деления на ноль
 except ZeroDivisionError:
 print("Ошибка: деление на ноль!")

Пример использования конструкции try-except-finally:

try:
 result = 10 / 0 # Попытка деления на ноль
 except ZeroDivisionError:
 print("Ошибка: деление на ноль!")
 finally:
 print("Блок finally выполнен всегда")

Лео решил испечь торт к своему дню рождения, но в процессе приготовления может возникнуть множество ошибок. Помоги Йорику написать программу, которая будет обрабатывать возможные ошибки при приготовлении торта.

Задачи

  • Создай программу, которая выполняет следующие шаги:
  • Делит количество ингредиентов на порции (возможная ошибка деления на ноль).
  • Принимает ввод от пользователя для количества порций (возможная ошибка ввода).
  • Используй конструкции try-except для обработки ошибок.
  • Используй конструкцию try-except-finally для выполнения кода, который должен выполниться в любом случае (например, сообщение о завершении программы).
