Условные операторы. if

Управляющие конструкции — это основа любой программы, которая позволяет разработчику определённым образом управлять ходом выполнения кода.

Оператор if в Python используется для выполнения блока кода, если определенное условие истинно. Это позволяет программе принимать решения на основе значений переменных или результатов выражений.

x = 10
 if x > 5:
 print("x больше 5")
