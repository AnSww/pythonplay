Операции над множеством

Добавление элементов: Метод add() используется для добавления нового элемента в множество.

numbers.add(6)

Удаление элементов: Метод remove() используется для удаления определенного элемента из множества. Если элемент не существует, будет сгенерировано исключение.

numbers.remove(3)

Проверка вхождения элемента: Оператор in используется для проверки наличия элемента в множестве.

print(4 in numbers) # Выведет True, так как 4 содержится в множестве numbers

Операции над множествами: Python предоставляет операции над множествами, такие как объединение (union), пересечение (intersection), разность (difference) и симметричная разность (symmetric_difference), которые позволяют выполнять операции над элементами двух множеств.

set1 = {1, 2, 3}
 set2 = {3, 4, 5}
 union_set = set1.union(set2) # Объединение множеств
 intersection_set = set1.intersection(set2) # Пересечение множеств
 difference_set = set1.difference(set2) # Разность множеств
 symmetric_difference_set = set1.symmetric_difference(set2) # Симметричная разность множеств.
